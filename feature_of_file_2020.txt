2020 엑셀 파일 설명

1. anime.csv
anime.csv에는 장르, 통계, 스튜디오 등 모든 애니메이션(17.562개의 다른 애니메이션)의 일반 정보가 포함되어 있습니다. 이 파일에는 다음과 같은 열이 있습니다:

: 애니, 평점, 장르, 영화 여부, 에피소드 개수, 작가, 스튜디오, 출처, 하트 횟수, 평점 10~1 개수

MAL_ID: 애니메이션의 내 애니멀리스트 아이디 (예: 1) _ 하위 파일들에 연동해야함
이름: 애니메이션의 전체 이름. (예: 카우보이 비밥)
점수: MyAnimelist 데이터베이스의 모든 사용자로부터 받은 애니메이션의 평균 점수. (예: 8.78)
장르: 이 애니메이션에 대한 쉼표로 구분된 장르 목록. (예: 액션, 어드벤처, 코미디, 드라마, 공상 과학, 공간)
영어 이름: 애니메이션의 전체 이름. (예: 카우보이 비밥)
일본어 이름 : 애니메이션의 일본어 풀네임 (예: カウボーイビバップ)
유형: TV, 영화, OVA 등(예: TV)
에피소드': 챕터 수 (예: 26)
방영일 : 방송일 (예: 1998년 4월 3일 ~ 1999년 4월 24일)
초연: 시즌 초연. (예: 1998년 봄)
프로듀서: 쉼표로 구분된 프로듀서 목록(예: 반다이 비주얼)
라이선스 제공자: 쉼표로 구분된 라이선스 제공자 목록(예: Funimation, Bandai Entertainment)
스튜디오: 쉼표로 구분된 스튜디오 목록(예: 선라이즈)
출처 : 만화, 라이트 노벨, 북 등 (오리지널 등)
듀레이션: 에피소드당 애니메이션의 듀레이션(예: 에피소드당 24분)
등급: 연령 비율(예: R - 17+ (폭력 및 비속어)
순위: 점수에 기반한 포지션(예: 28)
인기도 : 목록에 애니메이션을 추가한 사용자 수를 기준으로 한 순위 (예 39)
회원: 이 애니메이션의 "그룹"에 속한 커뮤니티 회원의 수. (예: 1251960)
즐겨찾기: 애니메이션을 "즐겨찾기"로 설정한 사용자 수 (예: 61,971명)
시청 : 애니메이션을 시청하는 사용자의 수 (예: 105808)
완료: 애니메이션을 완료한 사용자 수 (예: 718161)
On-Hold : 애니메이션을 보류 상태로 둔 사용자 수 (예: 71513)
탈락자 : 애니메이션을 탈락시킨 사용자의 수 (예: 26678)
Plan to Watch': 애니메이션을 볼 계획인 사용자의 수. (예: 329800)
Score-10': 10점을 획득한 사용자의 수. (예: 229170)
(10~1)

- episode, aired 깨지는 부분의 경우 xlsx로 열면 깨지지 않는다.
- 평점의 경우 다 본 평가자들에 의해 평가
- 일본어 제거

3. animelist.csv
animelist.csv는 사용자가 각각의 점수, 시청 상태, 시청한 에피소드 수와 함께 등록한 모든 애니메이션 목록을 가지고 있습니다. 
이 데이터 세트는 1억 9백만 행, 17.562개의 다른 애니메이션 및 325.772개의 다른 사용자를 포함합니다. 파일에는 다음과 같은 열이 있습니다:

user_id: 식별할 수 없는 임의로 생성된 사용자 ID입니다. _ 임의로 0~시작해서 차차 올라간다. 
anime_id: 애니메이션의 My Anemlist ID (예: 1). _ anime.csv의 id 참고. 
rating : 사용자가 부여한 1에서 10 사이의 점수. 사용자가 점수를 부여하지 않은 경우 0. (예: 10) ( = 평점) 
watching_status: 이 사용자의 애니메이션 목록에 있는 이 애니메이션의 상태 ID. (예: 2)(1~6까지)
watched_episodes : 사용자가 시청한 에피소드 수 (예: 24

- watching status 파일에서 상태 확인 가능
1: 최근 시청시작(currently watching)	2: 시청완료(completed)	3: 보는 중(on hold)		4. 하차(dropped)	5. 없음(아예 없다) 	6. 볼 계획( plan to watch)
 
4. rating_complete .csv
rating_complete.csv는 animelist.csv의 하위 집합입니다. 
이 데이터 집합은 사용자가 완전히 시청한 애니메이션만 고려하고(watching_status==2) 점수를 부여합니다(score!=0). 
이 데이터 집합에는 310.059명의 사용자가 16.872편의 애니메이션에 적용한 5700만 등급이 포함되어 있습니다. 이 파일의 열은 다음과 같습니다:

= 다 본 사람만 식별해서 rating 평균 매겨 둔 파일

user_id: 식별할 수 없는 임의로 생성된 사용자 ID입니다.
anime_id: - 이 사용자가 등급을 매긴 애니메이션의 내 애니멀리스트 ID.
rating: 이 사용자가 할당한 rating.
